# sitectl-omeka-classic

A [sitectl](https://sitectl.libops.io) plugin for Omeka Classic utilities. It works with the [LibOps Omeka Classic template](https://github.com/libops/omeka-classic).

The docs are available at [https://sitectl.libops.io/plugins/omeka-classic](https://sitectl.libops.io/plugins/omeka-classic).
